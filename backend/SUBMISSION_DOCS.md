# Quick Add to Hermes Eco

## Fastest Way (30 seconds)

### 1. Create `.hermes-eco.json` in your repo root:

\`\`\`json
{
  "name": "Your Resource Name",
  "type": "agent",
  "primary_category": "automation",
  "description": "What it does",
  "author": "your_github_username",
  "repository": "https://github.com/your/repo",
  "license": "MIT",
  "tags": ["tag1", "tag2"]
}
\`\`\`

### 2. Push to GitHub:

\`\`\`bash
git add .hermes-eco.json
git commit -m "Add to Hermes Eco"
git push
\`\`\`

### 3. Done! ✅ Auto-discovered within 24 hours
